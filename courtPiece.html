Project Overview

Court Piece and Shelem are two online card games with the same infrastructure in terms of code base.
I joined the project as a senior developer after the first version of these games got published and when the Court Piece had about 20k daily active users. So, my main task was to maitain the game by adding new features like game modes and liveops things. Besides, I started refactoring some parts of the code to make it more readable and extendable.
The main difference between these 2 games is in their core mechanics, and not in the meta-game. As my main tasks was to extend the meta-game, from here, I just mention Court Piece.

    After I jouned the team, we:
  1. - added conditional special offers to the shop.
  2. - added a seasonal liveops with a theme like Squid Game Netflix series, following the company's marketing campaign.
  3. - developed a competitive liveops mode called Myket League, following a campaign held by Myket, one of the markets where the Court Piece was published on it.
  4. - localized the game for international market, using a unity tool called I2 Localization.
  5. - added a purchasing feature(besides the IAP features related to market SDKs) using deeplink.
  6. - optimized the game's internet usage, by designing and developing a caching system, both in the client and the server sides.
  7. - implemented firebase
    8. Besides, I was responsible to prepare market android builds.
